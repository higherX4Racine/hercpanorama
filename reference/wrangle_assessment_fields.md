# Tidy assessment data into long format

Tidy assessment data into long format

## Usage

``` r
wrangle_assessment_fields(.panorama_data)
```

## Arguments

- .panorama_data:

  output from
  [`import_raw_report()`](https://higherx4racine.github.io/hercpanorama/reference/import_raw_report.md)

## Value

a tibble with fields that depend upon the suite of assessments
