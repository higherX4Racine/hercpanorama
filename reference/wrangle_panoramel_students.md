# Condense values that describe student identity into a single table

Condense values that describe student identity into a single table

## Usage

``` r
wrangle_panoramel_students(.panoramel_findings)
```

## Arguments

- .panoramel_findings:

  a dataframe produced by
  [`read_panoramel_spreadsheet()`](https://higherx4racine.github.io/hercpanorama/reference/read_panoramel_spreadsheet.md)

## Value

a data frame with student demographic information
