# hercpanorama: Wrangle data from the Panorama Student Information System

Student-level data from Panoram is export as wide-formatted csv files.
Each file has common identifying and demographic data, then many,
possibly unique, columns corresponding to scores on assessments or
responses to surveys. The names of these columns include information
about the type of data collected as well as when the data were
collected. There are also usually two columns for every assessment, a
raw score and a grade.
